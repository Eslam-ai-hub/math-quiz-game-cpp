# 🧮 C++ Math Quiz Game

A simple and interactive **C++ console-based Math Quiz Game** that helps players practice basic arithmetic operations through multiple levels and difficulties.

This project is part of the **AbouHadhood Platform** training track with **Dr. Mohamed Abou Hadhoud**.

---

## 📌 Features

- 🎯 Choose the number of questions (up to 100)
- 📊 Multiple difficulty levels:
  - Easy
  - Medium
  - Hard
  - Mix
- ➕➖✖️➗ Multiple operation types:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Mixed operations
- 🎨 Colored screen feedback:
  - 🟢 Green → Correct answer
  - 🔴 Red → Wrong answer (with alert sound)
- 📈 Final results summary:
  - Number of correct answers
  - Number of wrong answers
  - Pass / Fail result
- 🔁 Option to replay the quiz multiple times

---

## 🛠️ How It Works

1. The player selects:
   - Number of questions
   - Question difficulty level
   - Operation type
2. The game generates random math questions based on the selected options.
3. The player answers each question.
4. Immediate feedback is shown after every answer.
5. At the end of the quiz:
   - Results are calculated
   - Final status (Pass / Fail) is displayed
